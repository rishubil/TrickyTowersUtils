Tricky Towers Utils
===================

## Requirements

Git, Python 3.7 and Pipenv

## Installation

```
git clone https://github.com/rishubil/TrickyTowersUtils.git
pipenv install
```

## Configuration

Copy `config(example).ini` to `config.ini` and edit it.

## Run

```
pipenv run obsscript
```
